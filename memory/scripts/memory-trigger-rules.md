# 记忆搜索触发规则

## 🎯 自动触发关键词

当用户问题包含以下关键词时，**自动搜索记忆**：

### 时间相关
- "之前"、"刚才"、"昨天"、"今天"、"前天"、"上周"、"最近"
- "之前说的"、"昨天讨论的"、"今天聊的"
- "还记得"、"记得吗"、"还记得吗"

### 指代相关
- "那个..."、"这个..."、"哪些"
- "之前那个"、"刚才那个"
- "你说的"、"我说的"

### 回顾相关
- "怎么样了"、"进展"、"进度"
- "总结一下"、"回顾"、"复盘"
- "之前的问题"、"之前的决定"

### 项目相关
- "[项目名] + 怎么样/进度/问题"
- "小程序"、"批量上传"、"家庭共享"

---

## 🏠 房间选择逻辑

根据问题类型自动选择房间：

| 问题特征 | 搜索房间 |
|----------|----------|
| 技术、代码、Bug、开发 | work-dev |
| 需求、功能、产品、PRD | work-product |
| 反馈、意见、建议、满意 | life-feedback |
| 决定、选择、方案 | decisions |
| 待办、任务、计划 | todo |
| 生活、聊天、随便 | life-personal |
| 学习、知识、技能 | learning |
| 不确定 | 全部房间 |

---

## 📋 搜索流程

```
用户提问
  ↓
检测关键词 → 是否需要搜索？
  ↓ 是
分析问题类型 → 选择房间
  ↓
执行搜索 → mempalace search
  ↓
解析结果 → 提取 Top 3
  ↓
组织回答 → 结合记忆 + 上下文
```

---

## 💡 示例

**搜索：** `mempalace search "批量上传 功能" --room work-dev`
**回答：** 基于搜索结果回复
