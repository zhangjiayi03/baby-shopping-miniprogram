### 2. stats.js 月度趋势计算错误 ✅ 已修复

**文件:** `pages/stats/stats.js`  
**状态:** ✅ **已修复**（2026-04-13）

**原问题:** 柱状图高度计算逻辑存在 bug，每次循环都重新计算最大值。

- 添加了最小高度 10% 的保底值
