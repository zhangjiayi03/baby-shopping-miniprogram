## 测试前准备

### 1. 项目配置
```bash
# 1. 在微信开发者工具中打开 miniprogram 目录
# 2. 填写 app.json 中的 AppID
# 3. 在 app.js 中配置云环境 ID
# 4. 开通云开发，创建数据库集合 `records`
```

### 2. 云函数部署
```bash
# 在微信开发者工具中：
# 1. 右键 cloudfunctions/record 目录
# 2. 选择"上传并部署：云端安装依赖"
# 3. 等待部署完成
```

### 3. 数据库权限配置
在云开发控制台设置 `records` 集合权限：
- 创建：所有用户
- 查询：所有用户
- 更新：仅创建者
- 删除：仅创建者
