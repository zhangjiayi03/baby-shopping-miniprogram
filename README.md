# 宝宝购物记账小程序

一款专为新手爸妈设计的购物记账工具，帮助您轻松记录和管理宝宝用品的开支。

## 功能特点

- 📸 **截图识别** - 拍照/上传订单截图，自动识别商品名、价格、平台
- 🤖 **智能分类** - AI 自动识别商品类别（奶粉、尿布、玩具等）
- 👶 **宝宝档案** - 支持多宝宝管理，分别统计每个宝宝的花费
- 📊 **统计分析** - 日/周/月/年统计，分类占比，趋势分析
- 💰 **预算管理** - 设置月度预算，超支预警

## 技术栈

- 微信小程序原生开发
- 微信云开发（数据存储、云函数）
- 百度 OCR API（订单识别）

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/zhangjiayi03/baby-shopping-miniprogram.git
```

### 2. 导入微信开发者工具

1. 打开微信开发者工具
2. 导入项目，选择项目目录
3. AppID 使用测试号或填入自己的 AppID

### 3. 配置云开发

1. 开通云开发环境
2. 创建云环境（如 `cloud1-xxx`）
3. 修改 `app.js` 中的云环境 ID

### 4. 配置 OCR（可选）

如需使用截图识别功能：

1. 申请百度 OCR API Key
2. 创建 `cloudfunctions/ocr/config.js`：
   ```javascript
   module.exports = {
     BAIDU_OCR: {
       APP_ID: '你的AppID',
       API_KEY: '你的ApiKey',
       SECRET_KEY: '你的SecretKey'
     }
   }
   ```
3. 部署云函数

### 5. 运行项目

点击「编译」即可预览

## OCR 配置说明

本小程序支持三种 OCR 方案：

| 方案 | 费用 | 识别率 | 推荐度 |
|------|------|--------|--------|
| 百度 OCR | 500次/天免费 | 高 | ⭐⭐⭐⭐⭐ |
| 腾讯云 OCR | 1000次/月免费 | 高 | ⭐⭐⭐⭐ |
| 微信 OCR | 免费但有限制 | 中 | ⭐⭐⭐ |

## 隐私保护

- 所有数据存储在用户本地或用户自己的云开发环境
- 截图识别不保存敏感信息（手机号、地址、姓名等）
- 不上传任何隐私数据到第三方服务器
