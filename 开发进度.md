## 📊 数据计算逻辑

### 本月花费
```javascript
const thisMonthStart = new Date().setDate(1);
const spentThisMonth = records
  .filter(r => new Date(r.orderTime) >= thisMonthStart)
  .reduce((sum, r) => sum + parseFloat(r.price) * r.quantity, 0);
```

### 今日花费
```javascript
const today = new Date().toDateString();
const todaySpent = records
  .filter(r => new Date(r.orderTime).toDateString() === today)
  .reduce((sum, r) => sum + parseFloat(r.price) * r.quantity, 0);
```

### 环比对比
```javascript
const changePercent = ((today - yesterday) / yesterday) * 100;
// ↑ 红色（上涨） / ↓ 绿色（下降）
```

## ⚠️ 注意事项

### 1. Tab 图标
当前使用占位文件，需要替换为实际 PNG 图标：
- 建议尺寸：81x81px
- 格式：PNG（支持透明背景）
- 风格：线性图标（未选中） / 填充图标（选中）

### 2. 云函数
当前使用模拟数据，需要部署云函数：
- `record` - 记录 CRUD
- `stats` - 统计数据计算

### 3. 数据库
需要创建 `records` 集合并配置权限：
```javascript
// 权限规则
{
  "read": "auth.openid == doc.openid",
  "write": "auth.openid == doc.openid"
}
```
